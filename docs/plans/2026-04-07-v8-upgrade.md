# V8 Upgrade Implementation Plan

**Goal:** 将 `stock-assistant` 从 V7 升级到 V8，完成数据库扩容、指标预计算、同步扩展、历史回填与文档收口。

**Architecture:** 以 `docs/V8_架构设计与开发计划.md` 为准，把升级拆成“版本准备 -> 模型/迁移 -> 指标引擎 -> 同步扩展 -> 历史回填 -> 集成验证”六段。每段保持最小闭环：先补测试，再实现，再运行针对性验证，最后再进入下一段。

**Tech Stack:** Python 3.12+, FastAPI, SQLAlchemy 2.0, Alembic, SQLite, uv, pytest
