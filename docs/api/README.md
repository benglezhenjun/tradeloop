# API 契约 · OpenAPI

[`openapi.json`](./openapi.json) 是后端全部接口的机器可读契约（OpenAPI 3），由 FastAPI 从真实路由自动生成。

- **重新生成**：`cd backend && uv run python scripts/export_openapi.py`
- **在线查看**：本地启动后访问 `http://localhost:8000/docs`（Swagger UI）
- **防漂移**：测试 `tests/test_openapi_contract.py` 会校验本文件与当前代码一致——改了接口却忘了重新导出，CI 即报错。

`openapi.json` is the machine-readable contract (OpenAPI 3) for the whole backend API, auto-generated by FastAPI from the real routes. Regenerate with the command above; a contract test guards against drift in CI.
